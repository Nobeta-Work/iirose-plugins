# IIROSE Plugins

[IIROSE 蔷薇花园](https://www.iirose.com) 插件开发合集。

- **I@A** (`iirose-@all`)：简化 @ 全体成员方式

## 快速开始

### IIROSE @ALL

在 IIROSE 终端输入 `js` 后，粘贴下面任一链接。

在房间聊天框发送 `[@全体成员]` 即可触发。

### GitHub Raw

```text
https://raw.githubusercontent.com/Nobeta-Work/iirose-plugins/main/iirose-%40all/dist/bundle.js
```

### jsDelivr CDN

```text
https://cdn.jsdelivr.net/gh/Nobeta-Work/iirose-plugins@main/iirose-%40all/dist/bundle.js
```
